# CmdEdit Native App

This directory contains the actual CmdEdit macOS implementation:

- [`App`](/Users/zai28/dev/CmdEdit/CmdEdit/App): Swift package for the native overlay app
- [`ShellIntegration`](/Users/zai28/dev/CmdEdit/CmdEdit/ShellIntegration): `zsh` widget and keybinding
- [`Scripts`](/Users/zai28/dev/CmdEdit/CmdEdit/Scripts): build and install helpers

For installation and usage, start with the repo README at [`/Users/zai28/dev/CmdEdit/README.md`](/Users/zai28/dev/CmdEdit/README.md).

## Local Commands

Build:

```bash
bash Scripts/build.sh
```

Install:

```bash
bash Scripts/install.sh
```
